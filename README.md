# PHP Unit Converter

A simple unit converter web app built with HTML, CSS, and PHP that can convert between different units of measurement. It can convert units of length, weight, volume, area, temperature, and more. The user can input a value and select the units to convert from and to. The application will then display the converted value. This project is inspired by [roadmap.sh](https://roadmap.sh/projects/unit-converter).

## Features

This app can convert between different units of measurement for length, weight and temperature.

Supported conversions include:

- **Length**:

  - millimeter (mm)
  - centimeter (cm)
  - meter (m)
  - kilometer (km)
  - inch (in)
  - foot (ft)
  - yard (yd)
  - mile (mi)

- **Weight**:

  - milligram (mg)
  - gram (g)
  - kilogram (kg)
  - ounce (oz)
  - pound (lb)

- **Temperature**:
  - Celsius (°C)
  - Fahrenheit (°F)
  - Kelvin (K)

## Prerequisites

To run this web app, you’ll need:

- **PHP**: Version 7.4 or newer

## How to install

1. Clone the repository

   ```shell script
   git clone https://github.com/krisnaajiep/php-unit-converter.git
   ```

2. Change the current working directory

   ```shell script
   cd path/php-unit-converter-cli
   ```

3. Run the PHP built-in web server

   ```shell script
   php -S localhost:8000
   ```

4. Access the URL
   ```shell script
   http://localhost:8000
   ```
